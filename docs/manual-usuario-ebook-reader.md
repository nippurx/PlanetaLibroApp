# Manual de usuario del lector de libros digitales

Esta guía explica cómo leer y navegar por un libro digital en PlanetaLibro, personalizar su aspecto, retomar una lectura y compartir un pasaje.

## Abrir un libro

Al elegir la opción de lectura de un libro, PlanetaLibro abre el lector en una pantalla sin menús generales ni elementos que distraigan.

La primera vez se muestra el modo **Paginado**. Si el libro ya se había comenzado, el lector intenta recuperar el pasaje guardado. Un enlace compartido puede abrir directamente una ubicación determinada.

La barra negra con el logo y el nombre **PlanetaLibro.com** permanece visible en la parte superior para identificar el lector.

## Mostrar y ocultar los controles

Los controles se ocultan automáticamente para aprovechar el espacio de lectura.

Para mostrarlos u ocultarlos:

- En celular o tablet, toca una vez la zona central de la página.
- En una computadora, haz clic en la zona central del texto.
- Con teclado, pulsa `Escape` para mostrar los controles cuando estén ocultos.

Cuando un panel está abierto o un control tiene el foco del teclado, los controles permanecen visibles.

## Avanzar y retroceder

### En celular o tablet

- Toca el lateral derecho para avanzar una pantalla.
- Toca el lateral izquierdo para retroceder una pantalla.
- Desliza horizontalmente hacia la izquierda para avanzar.
- Desliza horizontalmente hacia la derecha para retroceder.
- También puedes usar los botones con flechas visibles en los laterales.

Una selección de texto, un desplazamiento vertical o una pulsación prolongada no deben cambiar de página.

### En una computadora

- Usa la flecha derecha, `Page Down` o la barra espaciadora para avanzar.
- Usa la flecha izquierda o `Page Up` para retroceder.
- También puedes hacer clic en los laterales o usar los botones con flechas.

## Elegir el modo de lectura

Abre **Preferencias** mediante el botón con el icono de texto y elige uno de estos modos:

- **Paginado:** presenta una pantalla de lectura por vez. Es el modo predeterminado.
- **Continuo:** permite desplazarse verticalmente como en una página web.

Al cambiar de modo, el lector intenta conservar visible el mismo pasaje.

## Usar el índice

1. Muestra los controles.
2. Pulsa el botón **Índice**.
3. Elige un capítulo o una sección.

El lector abre el destino elegido y comunica el capítulo actual. Para cerrar el índice sin cambiar de lugar, usa su botón de cierre o pulsa `Escape`.

## Personalizar la lectura

Abre **Preferencias** para modificar:

- Tema: **Claro**, **Sepia** u **Oscuro**.
- Fuente: **Serif**, **Sans** o **Accesible**.
- Tamaño del texto.
- Interlineado.
- Ancho del área de lectura y sus márgenes.

Los cambios se aplican al libro y se guardan en el dispositivo. Cuando el texto se redistribuye, el lector intenta mantener visible el pasaje que se estaba leyendo.

## Consultar el progreso

Al mostrar los controles, la barra inferior indica:

- La referencia de ubicación dentro del libro.
- El total publicado.
- Un porcentaje aproximado de lectura.

El número mostrado es una referencia del contenido publicado. No representa una página física fija: puede cambiar cuando se modifica el tamaño del texto, la fuente, los márgenes o la pantalla.

## Retomar una lectura

El lector guarda periódicamente el pasaje actual. Al volver a abrir el mismo libro, intenta continuar desde esa ubicación aunque hayan cambiado el tamaño de la pantalla o las preferencias visuales.

Si se abre un enlace que contiene una ubicación específica, esa ubicación tiene prioridad para esa apertura.

## Compartir una ubicación

Si no hay texto seleccionado:

1. Muestra los controles.
2. Pulsa el botón **Compartir**.
3. Elige una aplicación o contacto.

El mensaje utiliza el texto estándar de PlanetaLibro y un enlace a la ubicación visible. Si el dispositivo no ofrece el menú nativo para compartir, el lector copia el enlace al portapapeles.

## Compartir un texto seleccionado

Esta función permite compartir una cita del libro en lugar del mensaje estándar.

1. Mantén pulsada una palabra o arrastra el puntero sobre el texto.
2. Ajusta la selección hasta incluir la cita deseada.
3. Los controles se muestran y permanecen visibles mientras la cita está seleccionada.
4. Pulsa **Compartir**.
5. Elige la aplicación o el contacto de destino.

El mensaje incluye la cita seleccionada y un enlace corto a la parte del libro donde comienza. Si el menú nativo no está disponible, ambos se copian al portapapeles.

La selección debe comenzar y terminar dentro del contenido del libro. Si está vacía o incluye elementos externos al texto, el lector utiliza el mensaje estándar y la ubicación visible.

## Salir del lector

Muestra los controles y pulsa la flecha **Volver** de la barra superior. El lector guarda el progreso antes de abrir la ficha del libro.

## Usar el lector con teclado

| Acción | Tecla |
| --- | --- |
| Avanzar una pantalla | Flecha derecha, `Page Down` o barra espaciadora |
| Retroceder una pantalla | Flecha izquierda o `Page Up` |
| Cerrar un panel | `Escape` |
| Mostrar los controles ocultos | `Escape` |
| Recorrer botones y opciones | `Tab` y `Mayús` + `Tab` |
| Activar el elemento enfocado | `Enter` o barra espaciadora |

Las teclas de navegación no cambian de página mientras el foco está en un botón, enlace o campo que utiliza esas teclas.

## Accesibilidad

El lector ofrece:

- Controles visibles como alternativa a los gestos.
- Nombres accesibles y foco visible para botones y paneles.
- Navegación mediante teclado.
- Temas con contraste y texto adaptable al zoom del navegador.
- Compatibilidad con la preferencia del sistema para reducir el movimiento.
- Conservación de los textos alternativos incluidos en las imágenes del libro.
- Anuncios discretos al cambiar de capítulo o ubicación mediante una acción del usuario.

## Carga y resolución de problemas

### El lector muestra “Preparando el libro” o “Ajustando la página”

Espera mientras se cargan el contenido, las imágenes y las fuentes necesarias. El lector puede volver a ajustar la distribución al cambiar de orientación o preferencias.

### Un tramo siguiente no carga

Comprueba la conexión e intenta avanzar nuevamente. El contenido ya cargado y el progreso anterior permanecen disponibles siempre que el navegador pueda conservarlos.

### El libro no puede abrirse

La pantalla de error ofrece dos alternativas:

- Volver a la ficha del libro.
- Abrir el lector clásico.

## Mantenimiento de este manual

Para que este archivo pueda publicarse posteriormente como HTML:

- Mantener un único título principal (`h1`) y una jerarquía ordenada de encabezados.
- Usar enlaces internos relativos y texto descriptivo para cada enlace.
- Describir las acciones por su nombre visible, no solamente por la forma de su icono.
- Incluir texto alternativo descriptivo si se agregan capturas.
- Evitar instrucciones dependientes de un dispositivo cuando exista una alternativa equivalente.
- Actualizar `version` y `last_updated` cuando cambie el comportamiento documentado.
- Agregar cada nueva función visible al contenido y a la tabla inicial antes de considerarla terminada.
